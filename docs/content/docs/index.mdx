**vimstyle-ui** is a collection of React components and hooks for building keyboard-first interfaces in Next.js. Think vim keybindings meets modern UI — `j`/`k` to navigate, `/` to search, `Enter` to act, `Esc` to go back.

The design language is dark-mode zinc with violet accents. Every component ships as a plain `.tsx` file you own and modify. There is no npm package to install — you clone the repo and the code is yours.

## Quick start

```bash
git clone https://github.com/cellexec/vimstyle-ui.git
cd vimstyle-ui
bun install
bun run dev
```

The dev server starts with the docs site and a live preview of every component.

## What's included

- **11 components** — Full-page layouts, navigable lists, search bars, keyboard hints, dialogs, and overlays.
- **4 hooks** — Vim-style keyboard navigation, fuzzy filtering, mouse interaction guards, and auto-scroll.
- **Claude Code skill** — An AI skill file so Claude understands the full pattern system when working on your fork.
