**vimstyle-ui** is a collection of React components and hooks for building keyboard-first interfaces in Next.js. Think vim keybindings meets modern UI — `j`/`k` to navigate, `/` to search, `Enter` to act, `Esc` to go back.

The design language is dark-mode zinc with violet accents. Every component ships as a plain `.tsx` file you own and modify. There is no npm package to install — you clone the repo and the code is yours.

## Quick start

```bash
git clone https://github.com/cellexec/vimstyle-ui.git
cd vimstyle-ui
bun install
bun run dev
```

The dev server starts with the docs site and a live preview of every component.

## Self-contained docs

This documentation site is built with the same components it documents. The sidebar uses `useNavHints` for space-motion navigation. The demo previews render the actual registry components. Changing a file in `registry/` changes both the library and the docs — fork the repo, modify a component, run `bun run dev`, and see the result immediately.

Press `Space` right now to try space-motion navigation on this sidebar.

## What's included

- **12 components** — Full-page layouts, navigable lists, search bars, keyboard hints, dialogs, overlays, and hint labels.
- **5 hooks** — Vim-style keyboard navigation, fuzzy filtering, mouse interaction guards, auto-scroll, and space-motion hints.
- **3 layouts** — Pre-wired page templates: list, split-panel, and command palette.
- **Claude Code skill** — An AI skill file so Claude understands the full pattern system when working on your fork.
