These demos show the complete vimstyle-ui system in action — layouts, components, and hooks working together in realistic scenarios. Each demo includes a device size toggle to preview responsiveness at mobile, tablet, and desktop widths.

## Available Demos

| Demo | Layout | Description |
|---|---|---|
| [Projects](/docs/demos/projects) | ListLayout | Project management page with grouped items, actions, and delete confirmation. |
| [Mail Client](/docs/demos/mail-client) | SplitLayout | Email client with list + preview pane. |
| [Command Palette](/docs/demos/command-palette) | CommandPalette | Floating search overlay triggered by Ctrl+K. |

## Interacting with demos

All demos are fully interactive:

- **Keyboard**: Use `j`/`k` to navigate, `/` to search, `Enter` to select, `Esc` to go back
- **Mouse**: Click and hover work naturally alongside keyboard
- **Device toggle**: Switch between mobile (375px), tablet (768px), and desktop to see how layouts adapt
