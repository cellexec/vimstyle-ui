title: Components
description: All vimstyle-ui components at a glance.
---

vimstyle-ui ships 20 components. Each is a single `.tsx` file with no internal abstractions.

| Component | Description |
|---|---|
| [VimPage](/docs/components/vim-page) | Full-page layout with header, content area, and bottom keyboard hints bar. |
| [VimList](/docs/components/vim-list) | Navigable list panel combining a search bar with a scrollable item area. |
| [SearchBar](/docs/components/search-bar) | Fuzzy search input with magnifying glass icon and `/` keyboard hint. |
| [FuzzyText](/docs/components/fuzzy-text) | Inline text renderer that highlights fuzzy-matched characters. |
| [KbdHint](/docs/components/kbd-hint) | Small inline badge displaying a keyboard shortcut. |
| [KbdButton](/docs/components/kbd-button) | Action button with an embedded keyboard shortcut that registers a global listener. |
| [ListItem](/docs/components/list-item) | Selectable/editable list row with three visual states. |
| [SectionHeader](/docs/components/section-header) | Sticky group label for sectioned lists. |
| [FocusOverlay](/docs/components/focus-overlay) | Near-full-screen overlay panel for immersive content. |
| [ConfirmDialog](/docs/components/confirm-dialog) | Centered confirmation modal with Enter/Esc keyboard support. |
| [SaveDiscardDialog](/docs/components/save-discard-dialog) | Three-action modal for unsaved changes: Save, Discard, Cancel. |
| [HintLabel](/docs/components/hint-label) | Label that highlights the unique hint prefix during space-motion navigation. |
| [NavIndicator](/docs/components/nav-indicator) | Floating indicator showing space-motion hint mode state. |
| [YankIndicator](/docs/components/yank-indicator) | Floating indicator for yank mode with copy confirmation. |
| [Yankable](/docs/components/yankable) | Wrapper that marks content as copyable with a small clipboard icon. |
| [VimSidebar](/docs/components/vim-sidebar) | Full sidebar with j/k navigation, space-motion hints, collapsible sections, and drag-to-resize. |
| [EscapeBlur](/docs/components/escape-blur) | Global Escape handler that blurs any focused input. |
| [Tooltip](/docs/components/tooltip) | Hover tooltip with dynamic positioning via getBoundingClientRect. |
| [LogTerminal](/docs/components/log-terminal) | Terminal-style log viewer with line variants and smart auto-scroll. |
| [NotificationPanel](/docs/components/notification-panel) | Full notification panel with vim navigation, fuzzy search, date grouping, and actions. |
